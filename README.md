# socket-chat-client

## Usage

Rename `.env.sample` to `.env`

Then:

```bash
yarn
yarn start
```

## Features & Tech stack

- bootstrapped with `create-react-app`
- `typescript`
- tested with `jest` & `testing-library/react`
- using native `WebSocket` & `fetch`
- `scss` powered `css modules`
- custom simple `notifications`
